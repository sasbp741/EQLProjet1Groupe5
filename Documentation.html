DOCUMENTATION DE L'ANNUAIRE EQL

Présentation

Objectif

L'objectif de l'annuaire de l'EQL est de regrouper l'ensemble des stagiaires ayant suivi une formation à l'école, au sein d'une base de données commune.

Pour chaque stagiaire, on y retrouve les informations suivantes :

  • son nom ;
  • son prénom ;
  • son département de résidence ;
  • sa promotion
  • l'année de sa promotion.

Interface

L'interface du logiciel se veut intuitive et se compose de 3 panneaux :

  • une fenêtre principale, contenant la liste des stagiaires et le panneaux recherche avancée;
  • une fenêtre d'ajout et de modification d'un stagiaire ;
  • une fenêtre d'export de l'annuaire en document PDF.

Droits d'accès et fonctionnalités

Selon la personne utilisant l'annuaire EQL, il existe 2 modes donnant accès à différentes fonctionnalités :

  • le mode utilisateur (voir passage au mode Utilisateur);
  • le mode administrateur (voir passage au mode Administateur).

- Mode utilisateur

Accessible par défaut, ce mode offre la possibilité d'utiliser les fonctionnalités suivantes :

  • Consulter l'annuaire;
  • Rechercher un stagiaire;
  • Ajouter un stagiaire;
  • Exporter tout ou une partie de l'annuaire en PDF.

- Mode administrateur

Ce mode donne accès à l'ensemble des fonctionnalités du mode utilisateur, ainsi que les suivantes :

  • Modifier un stagiaire;
  • Supprimer un stagiaire.
